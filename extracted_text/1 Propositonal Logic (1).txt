вҖў Extensibility: Users can define new control structures and language features, making it highly adaptable.
вҖў Influence: It became the dominant language for symbolic AI research for decades.

Inference

Knowledge Bases
How can an AI agent store, organize, and reason with everything it knows about the 
world?
Definition. A knowledge base рқҗҫрқҗө is a set of sentences
рқҗҫрқҗө = рқҗҙ1, рқҗҙ2, вҖҰ , Aрқ‘ӣ
These sentences are typically (conceptually) connected by logical conjunctions:
рқҗҫрқҗө = рқҗҙ1 вҲ§ рқҗҙ2 вҲ§ вӢҜ вҲ§ Aрқ‘ӣ.
Each sentence рқҗҙрқ‘– represents a fact or assertion about the world, expressed in a 
formal logic language.

Ultimately, our goal is to infer (discover) new sentences that logically follow from a 
knowledge base.
Knowledge Base Entailment and Inference
Definition. In propositional logic, a sentence рқ‘Ҷ is entailed by a knowledge base рқҗҫрқҗө, 
written as
рқҗҫрқҗө вҠЁ рқ‘Ҷ
if and only if S is true in every model where all sentences in рқҗҫрқҗө are true.
Entailment tells us what must be true given the facts and rules in the рқҗҫрқҗө; it is the foundation 
for logical reasoning in AI systems.
But, what else can we conclude from our рқҗҫрқҗө?

Knowledge Base Operations

A rule-based system has the following components:
вҖў Facts: A database of true statements (known information).
вҖў Rules: вҖңIf-thenвҖқ statements that encode domain knowledge.
вҖў Inference Engine: Algorithms to apply rules to facts to derive new knowledge.
There are two main approaches 
in rule-based systems:
вҖў Data-driven reasoning with Forward Chaining.
вҖў Goal-driven reasoning with Backward Chaining.

Forward Chaining
FORWARD_CHAINING(KB, query):
    facts = initial_facts
    if query in facts: return True
   new_facts = True
   while new_facts:
        new_facts = False
        for each rule in KB:
            if not rule.conditions_satisfied_by(facts):
               continue
            if rule.conclusion in facts:
               continue
            add rule.conclusion to facts
           new_facts = True
           if rule.conclusion == query: return True
   return False
Find rules whose conditions 
match current facts
A new fact has been derived 
when all вҖҳconditionsвҖҷ are 
satisfied by database facts.
(Data-driven reasoning.)
Iterate exhaustively while 
new facts are inferred.

Backward Chaining
BACKWARD_CHAINING(KB, goal):
    if goal in facts: return True
    for each rule in KB:
        if rule.conclusion == goal:
            if PROVE_PREMISES(rule.conditions):
                return True
   return False
PROVE_PREMISES(premises):
    for each premise in premises:
       if not BACKWARD_CHAINING(KB, premise):
           return False
   return True
Think of the goal as being the root of a tree. Backward chaining is an inference technique that starts with a goal and 
builds that tree downward using the rules set. Ultimately, if the leaves of that tree are all in the known set of facts, 
we have deduced our goal from the knowledge base.

Problem. Prove that there are infinitely many prime numbers.
Can we prove this statement directly?
Doing so would require that we list or

Proof by contradiction.

a value рқ‘Ғ = рқ‘қ1 вӢ… рқ‘қ2 вӢ… вҖҰ вӢ… рқ‘қрқ‘ӣ + 1. Clearly, рқ‘Ғ is not divisible by any prime рқ‘қрқ‘– (as рқ‘Ғ % рқ‘қрқ‘– = 1 for all 
рқ‘–). 
вҲҺ

Soundness (of Propositional Logic)
Definition. An argument is sound if it 
is valid and all premises are actually true.Our general definition:
Definition. An inference system is sound if and only if for all knowledge bases рқҗҫрқҗө and all sentences 
рқңҷ,
рқҗҫрқҗө вҠў рқңҷ вҮ’ рқҗҫрқҗө вҠЁ рқңҷ
(If рқңҷ is derivable from рқҗҫрқҗө using the inference rules, then рқңҷ is semantically entailed by рқҗҫрқҗө.)
Soundness is a guarantee that our inference system never leads us astray: every derivable conclusion is logically 
valid.
Without a sound inference system, we cannot trust our proofs since we might derive рқңҷ from ~рқңҷ.

Completeness (of Propositional Logic)
Definition. An inference system is complete if and only if for all knowledge bases рқҗҫрқҗө and all 
sentences рқңҷ,
рқҗҫрқҗө вҠЁ рқңҷ вҮ’ рқҗҫрқҗө вҠў рқңҷ
(If рқңҷ is semantically entailed by рқҗҫрқҗө, then рқңҷ is derivable from рқҗҫрқҗө using the inference rules.)
Completeness is a guarantee that our inference system can find all valid conclusions (i.e., no 
вҖҳmissingвҖҷ logical consequences).
Without a complete inference system, some valid conclusions might be unprovable (since we would 
miss important logical consequences).

Suppose we have an inference system that only

Theorem. Resolution for propositional logic is both sound and complete.

Theorem. If рқҗҫрқҗө вҲ§ ~рқңҷ вҠўвҠҘ by Resolution, then рқҗҫрқҗө вҠЁ рқңҷ.
Proof. Suppose Resolution derives вҠҘ from рқҗҫрқҗө вҲ§ ~рқңҷ. 
Define two clauses рқҗ¶1 = рқҗҝ вҲЁ рқҗҙ  and рқҗ¶2 = ~рқҗҝ вҲЁ рқҗө  that are both satisfiable and also let рқ‘… = рқҗҙ вҲЁ рқҗө  be 
their resolvent.
Let рқ‘Җ be a model that satisfies both рқҗ¶1 and рқҗ¶2. Then рқ‘Җ must assign some truth value to рқҗҝ.
If рқ‘Җ рқҗҝ = рқ‘“рқ‘Һрқ‘ҷрқ‘ рқ‘’, then since рқ‘Җ satisfies рқҗ¶1 = рқҗҝ вҲЁ рқҗҙ  , we must have 
рқ‘Җ рқҗҙ =  рқ‘Ўрқ‘ҹрқ‘ўрқ‘’, which again makes рқ‘Җ рқ‘… = рқ‘Җ рқҗҙ вҲЁ рқҗө = рқ‘Ўрқ‘ҹрқ‘ўрқ‘’.
First, we establish that the resolution rule is truth-preserving.
Therefore, рқ‘… is satisfiable whenever both parent clauses are satisfiable.
Since resolution preserves satisfiability, any clause derived through resolution can only be satisfiable if the 
original clauses are satisfiable. Therefore, if we derive вҠҘ (the empty clause, which is always unsatisfiable), from 
рқҗҫрқҗө вҲ§ ~рқңҷ, this means рқҗҫрқҗө вҲ§ ~рқңҷ must be unsatisfiable.
That is, there is no model that makes both рқҗҫрқҗө  and ~рқңҷ simultaneously true. Thus, in every model where рқҗҫрқҗө is 
true, ~рқңҷ must be false. Hence, рқңҷ must be true. It follows рқҗҫрқҗө вҠЁ рқңҷ. вҲҺ
Sound
If рқ‘Җ рқҗҝ =  рқ‘Ўрқ‘ҹрқ‘ўрқ‘’, then since рқ‘Җ satisfies рқҗ¶2 = ~рқҗҝ вҲЁ рқҗө , we must 
have рқ‘Җ рқҗө =  рқ‘Ўрқ‘ҹрқ‘ўрқ‘’, which makes рқ‘Җ рқ‘… = рқ‘Җ рқҗҙ вҲЁ рқҗө = рқ‘Ўрқ‘ҹрқ‘ўрқ‘’.

Theorem. If рқҗҫрқҗө вҠЁ рқңҷ, then Resolution derives вҠҘ from  рқҗҫрқҗө вҲ§ ~рқңҷ.
Proof. Suppose рқҗҫрқҗө вҠЁ рқңҷ. Then рқҗҫрқҗө вҲ§ ~рқңҷ is unsatisfiable.
Observe that for propositional logic there are finitely many distinct clauses. Thus, we know that 
Resolution terminates with a finite clause set рқ‘Ҷ.
Therefore, Resolution must derive вҠҘ.вҲҺ
Since Resolution did not derive вҠҘ, the final clause set рқ‘Ҷ must be satisfiable. We therefore can construct 
a truth assignment that satisfies all clauses in рқ‘Ҷ:
вҖў For each propositional variable рқ‘ғ, if there exists a unit clause рқ‘ғ in рқ‘Ҷ, assign рқ‘ғ = рқ‘Ўрқ‘ҹрқ‘ўрқ‘’;
вҖў If there exists a unit clause ~рқ‘ғ in рқ‘Ҷ, assign рқ‘ғ = рқ‘“рқ‘Һрқ‘ҷрқ‘ рқ‘’;
вҖў Otherwise, assign рқ‘ғ arbitrarily.
Recall the Resolution procedure begins with рқҗҫрқҗө вҲ§ ~рқңҷ. Since рқ‘Ҷ contains all clauses from the CNF 
conversion of рқҗҫрқҗө вҲ§ ~рқңҷ (as well as possibly other derived clauses), the truth assignment we 
constructed also satisfies рқҗҫрқҗө вҲ§ ~рқңҷ. This is a contradiction.
Complete
